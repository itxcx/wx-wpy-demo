# wx-wpy-demo
一些wx小程序官网API项目的demo，改造成wepy版本。用来熟悉最新微信小程序和wepy框架新知识。

# 项目介绍
1. weui-wpydemo是weui的改造版本，项目UI参考样本。
2. wxcmpt-wpydemo是小程序组件改造版本，项目组件参考样本。
